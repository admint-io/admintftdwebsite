FAQs

We have developed the faq and question-accordion component for admint.io/learn, which matches the design currently displayed on the live site.

How to use?

To use the faq and question-accordion components you need to import lib.ftd inside your .ftd file, as shown below:
-- import: admint.io/lib
Lang:
ftd

Once you import the lib inside your .ftd file, use the faq and question-accordion component by adding all required attributes of it.

You can also use question-accordion component if you want to show a accordion like UI on your .ftd pages. Just add title and body to it. Check second question-accordion component in below code snippet.
